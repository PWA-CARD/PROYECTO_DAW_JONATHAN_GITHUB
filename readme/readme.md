# Centro de Musculación y Acondicionamiento Físico – Prototipo Estático (Bootstrap 5)
Proyecto Intermodular – Desarrollo de Interfaces Web  
Autor: **Jonathan Martínez Pagán**

Este repositorio contiene el **prototipo estático** del proyecto de gestión de reservas de:
“Centro de Musculación y Acondicionamiento Físico”.  
Esta versión NO incluye PHP, base de datos ni backend.  
Su objetivo es mostrar la interfaz pública y privada básica utilizando **HTML + CSS + Bootstrap 5**, cumpliendo los requisitos de la primera fase del proyecto.

## Tecnologías utilizadas
- HTML5

---

## Páginas incluidas

### **index.html – Landing page (zona pública)**
Incluye:
- Navbar
- Carrusel de imágenes
- Cards informativas
- Paginación
- Footer

### **login.html – Inicio de sesión**
Componentes:
- Formulario responsive
- Breadcrumb
- Enlaces a registro / recuperar contraseña

### **dashboard.html – Panel del usuario**
Incluye:
- Avatar
- Datos del usuario
- Accesos rápidos mediante cards
- Zona administrativa ilustrativa

### **reservas.html – Reservar clases**
Incluye:
- Listado de clases
- Aforo y plazas libres
- Badges de estado (Disponible, Completa, Apuntado)
- Botón de Reservar (visual)
- Paginación
